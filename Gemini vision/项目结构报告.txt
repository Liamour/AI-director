# AI 世纪导演 项目结构报告
## 项目基本信息
- 项目名称：AI故事板导演工具
- 技术栈：Next.js 14 + React 18 + TypeScript + TailwindCSS + Tauri 2 + Zustand
- 主要功能：AI驱动的剧本创作、分镜生成、项目管理工具
- 运行模式：支持Web浏览器预览和Tauri桌面应用两种模式

## 核心功能模块说明
1. **灵感奇点模块**：用户输入创意灵感，选择风格标签
2. **剧本导入模块**：支持导入本地剧本文件（.txt/.md/.json）
3. **项目创建模块**：创建本地项目目录结构（仅Tauri桌面端可用）
4. **AI生成模块**：基于用户输入生成完整剧本和分镜
5. **导演控制台**：可视化展示生成的分镜、脚本、人物设定

## 技术架构说明
- 状态管理：使用Zustand进行全局状态管理，定义在`src/store/scriptStore.ts`
- UI框架：使用TailwindCSS + Framer Motion实现动效和样式
- 桌面端能力：通过Tauri API实现本地文件系统访问、目录操作
- 路由架构：使用Next.js Pages Router（已从App Router迁移，避免水合错误）

## 关键文件说明
1. **src/pages/index.tsx**：主页面，包含灵感输入、风格选择、项目创建模态框
2. **src/store/scriptStore.ts**：全局状态，管理项目数据、生成状态、用户选择
3. **src/shared/lib/tauri-fs.ts**：Tauri API封装，包含环境检测，在浏览器环境自动降级
4. **src/features/director-board/ui/DirectorLayout.tsx**：导演控制台主界面
5. **next.config.js**：配置了静态导出和Tauri相关依赖忽略

## 运行说明
- 开发模式：`npm run dev` 启动在 http://localhost:3000
- 构建模式：`npm run build` 导出静态文件到`dist`目录，用于Tauri打包
- 桌面端：通过Tauri打包后可以访问本地文件系统，实现项目创建和文件导入功能
